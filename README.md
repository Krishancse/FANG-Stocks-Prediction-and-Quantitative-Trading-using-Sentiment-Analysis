# FANG Stocks Prediction and Quantitative Trading using Sentiment Analysis


## Overview

This project focuses on predicting the stock prices of FANG (Facebook, Amazon, Netflix, Google) companies using sentiment analysis techniques. Sentiment analysis involves extracting insights from textual data, such as news articles, social media posts, and financial reports, to gauge market sentiment and make trading decisions.

The main objectives of this project include:

- Collecting and preprocessing relevant textual data sources.
- Performing sentiment analysis on the collected data to gauge market sentiment towards FANG stocks.
- Developing predictive models for FANG stock prices based on sentiment analysis results.
- Implementing quantitative trading strategies based on the predictive models.

## Features

- Data Collection: The project includes scripts to collect textual data from various sources such as financial news websites, social media platforms (Twitter, Reddit), and other relevant sources.
- Text Preprocessing: Textual data undergo preprocessing steps including tokenization, stop word removal, and stemming to prepare it for sentiment analysis.
- Sentiment Analysis: Utilizes natural language processing (NLP) techniques to analyze the sentiment of collected textual data towards FANG stocks.
- Predictive Modeling: Implements machine learning algorithms to predict FANG stock prices based on sentiment analysis results and historical price data.
- Quantitative Trading Strategies: Develops and backtests quantitative trading strategies using the predictive models to generate buy/sell signals for FANG stocks.

## Requirements

- Python 3.x
- Libraries: pandas, numpy, scikit-learn, nltk, tweepy (for Twitter data collection), praw (for Reddit data collection), and any additional libraries as specified in the requirements file.

1. Data Collection:
   - Run the data collection scripts to gather textual data from various sources.
   - Configure API keys and access credentials as necessary (e.g., Twitter API keys, Reddit API keys).

2. Text Preprocessing:
   - Preprocess the collected textual data to remove noise and prepare it for sentiment analysis.

3. Sentiment Analysis:
   - Perform sentiment analysis on the preprocessed textual data using NLP techniques.

4. Predictive Modeling:
   - Train machine learning models to predict FANG stock prices based on sentiment analysis results and historical price data.

5. Quantitative Trading:
   - Develop and backtest quantitative trading strategies using the predictive models to generate trading signals for FANG stocks.
